
テンプレ init

```
$ npx @docusaurus/init@latest init [name] [template]
```

2021/08/27

[Docusaurusu themes](https://docusaurus.io/docs/api/themes) を読む限り

基本 template は classic を使ったほうが良さそう


```
$ npx @docusaurus/init@latest init hogehoge classic
```
